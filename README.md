# Gallery API

A secure backend built with **Node.js** and **Express** that proxies requests to the Unsplash API, keeping the access jey hidden from frontend applications.

## Features

- Secure access to the Unsplash API
- RESTful API design
- Environment-based configuration
- Production-ready project structure

## Tech Stack

- Node.js
- Express
- Unsplash API
